tengine的安装和使用

由于官方的 Nginx 缺乏一些常用特性。比如：

  1. 对于负载均衡，nginx 不支持 sticky session 方式。如果使用 ip hash, 也可能会负载不均衡。
  2. 不支持对后台服务的health check。

因此，服务器上决定使用 Tengine 来取代 Nginx 官方版。
